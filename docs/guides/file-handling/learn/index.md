# Learn: File Handling

Four short tutorials that build up your working knowledge of reading and writing files in Python. Each one is a Jupyter notebook — read it in the browser, and edit and run any code cell directly on the page.

The tutorials build on each other, so working through them in order is the fastest path. Each takes roughly fifteen to twenty minutes.

## The sequence

1. **[Reading files](01-reading-files.ipynb)** — `open`, the `with` statement, reading text line by line and all at once.
2. **[Writing files](02-writing-files.ipynb)** — write modes, appending, and getting written content to disk reliably.
3. **[Working with paths](03-working-with-paths.ipynb)** — `pathlib` for paths that work the same on every operating system.
4. **[CSV and JSON files](04-csv-and-json-files.ipynb)** — the two file formats you'll meet most often, and the modules that handle them.

## Before you start

You'll get more from these if you're already comfortable with Python strings, lists, and dictionaries. The [String processing](../../string-processing/) and [Data structures](../../data-structures/) guides cover what you need.

You don't need Python installed locally — every tutorial runs in your browser. If you'd rather run them on your own machine, each notebook has a download button in the top-right corner.
