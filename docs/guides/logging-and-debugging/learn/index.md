# Learn: Logging and Debugging

Four short tutorials that build up your working knowledge of Python's `logging` module and the `pdb` debugger. Each one is a Jupyter notebook — read it in the browser, and edit and run any code cell directly on the page.

The tutorials build on each other, so working through them in order is the fastest path. Each takes roughly fifteen to twenty minutes.

## The sequence

1. **[Your first log message](01-your-first-log-message.ipynb)** — getting `logging` configured and emitting your first messages.
2. **[Log levels and formatting](02-log-levels-and-formatting.ipynb)** — `DEBUG`, `INFO`, `WARNING`, `ERROR`, `CRITICAL`, and shaping the output.
3. **[Logging to files](03-logging-to-files.ipynb)** — handlers, rotation, and getting log output off stdout.
4. **[Debugging with pdb](04-debugging-with-pdb.ipynb)** — `breakpoint()`, stepping, inspecting state, and post-mortem debugging.

## Before you start

You'll get more from these if you're already comfortable with Python functions and modules. The [Functions](../../functions/) and [Error handling](../../error-handling/) guides are good warm-ups.

You don't need Python installed locally — every tutorial runs in your browser. If you'd rather run them on your own machine, each notebook has a download button in the top-right corner.
