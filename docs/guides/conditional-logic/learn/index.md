You'll get the most out of this if you're already comfortable with Python variables, numbers, and comparison operators. If that's still new, the opening sections of the [Data structures](../../data-structures/) guide cover the basics.

You don't need Python installed locally — the tutorial runs in your browser. If you'd rather run it on your own machine, the notebook has a download button in the top-right corner.
