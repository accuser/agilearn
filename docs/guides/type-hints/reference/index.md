# Reference: Type hints

Quick-lookup pages — short, dense, designed to skim. Start with [Learn](../learn/) if you're here to understand the topic for the first time.

## Pages in this section

- **[Built-in generic types](built-in-generic-types.md)** — annotating `list`, `dict`, `set`, `tuple`, and the rest with their element types. Also: `Iterable`, `Sequence`, `Mapping`, and when the abstract forms buy you something over the concrete ones.
- **[mypy cheatsheet](mypy-cheatsheet.md)** — running the type-checker, interpreting common errors, suppressing false positives with `# type: ignore`, and the handful of config options that matter.
